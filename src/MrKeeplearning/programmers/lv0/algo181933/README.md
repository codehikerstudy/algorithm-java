# flag에 따라 다른 값 반환하기

[문제 바로 가기](https://school.programmers.co.kr/learn/courses/30/lessons/181933)

## Solution1

```java
public class Solution {
    public int solution(int a, int b, boolean flag) {
        int answer = flag ? a + b : a - b;
        return answer;
    }
}
```
